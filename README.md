# Tambike PH

Motorcycle-only community platform for event discovery, partner businesses, event registration, QR attendance check-in, organizer dashboards, admin approvals, and raffle/giveaway workflows.

## Theme

- Primary: black
- Secondary/accent: neon pink
- Style: modern motorcycle night ride / street meet

## Scripts

```bash
pnpm install
pnpm test
pnpm run build
pnpm run dev -- --host 127.0.0.1
```

## MVP Screens Included

- Home/landing section
- Event discovery
- Business directory
- QR event pass preview
- Organizer dashboard
- Admin approval dashboard
